# AI Cipher Decoder ‚Äî Backend

Python/FastAPI backend for the AI Cipher Decoder project. Hosts the cipher engine, the cryptanalysis solvers, and the chatbot orchestration layer that powers the frontend.

Pairs with the `ai-cipher-decoder-frontend` repo, which provides the UI. This backend can also be run and tested entirely on its own via its API endpoints.

## What this backend does

1. **Cipher engine** (`app/ciphers/`) ‚Äî deterministic encrypt/decrypt for each supported cipher when the key is known: Caesar, Atbash, Affine, Vigen√®re, Hill, Nihilist, Baconian, Porta, Pollux, Morbit, Fractionated Morse, and keyed Aristocrat/Patristocrat.
2. **Cryptanalysis solvers** (`app/solvers/`) ‚Äî cracks ciphertext *without* a known key using frequency analysis, quadgram scoring, and hill-climbing/simulated annealing search:
   - Aristocrat/Patristocrat substitution solver
   - Vigen√®re solver (Kasiski examination + Index of Coincidence)
   - Xenocrypt (Spanish-language) solver
3. **Cryptarithm solver** (`app/solvers/cryptarithm_solver.py`) ‚Äî solves letter-arithmetic puzzles (e.g. `SEND + MORE = MONEY`) using constraint-satisfaction search over possible letter-to-digit assignments, rather than language-frequency techniques.
4. **Chatbot orchestration** (`app/chatbot/`) ‚Äî wraps the cipher and solver functions as tools the Anthropic API can call, so a user can type something like "decrypt this Caesar cipher: KHOOR ZRUOG" in plain English and get back both the answer and an explanation of the steps.
5. **REST API** (`app/routes/`) ‚Äî the endpoints the frontend calls to hit all of the above.

## Setup

```bash
git clone <this-repo-url>
cd ai-cipher-decoder-backend

python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

pip install -r requirements.txt

cp .env.example .env
# then open .env and add your ANTHROPIC_API_KEY

uvicorn app.main:app --reload
```

The API will be running at `http://localhost:8000`. Interactive docs (auto-generated by FastAPI) are available at `http://localhost:8000/docs`.

## Environment variables

| Variable | Description |
|---|---|
| `ANTHROPIC_API_KEY` | API key used by the chatbot orchestration layer to call Claude |

## API overview

| Endpoint | Method | Description |
|---|---|---|
| `/encrypt` | POST | Encrypt plaintext with a given cipher + key |
| `/decrypt` | POST | Decrypt ciphertext with a given cipher + key |
| `/solve` | POST | Auto-crack ciphertext with no key (Aristocrat, Vigen√®re, Xenocrypt) |
| `/cryptarithm` | POST | Solve a letter-arithmetic puzzle (e.g. `SEND + MORE = MONEY`), returns the digit mapping |
| `/chat` | POST | Send a natural-language message; chatbot picks the right tool and responds |

Full request/response schemas are viewable at `/docs` once the server is running.

## Running tests

```bash
pytest app/tests/
```

## Data dependencies

The solvers require frequency data to score candidate decryptions:

- `app/data/english_quadgrams.txt` ‚Äî English quadgram frequencies, used by the Aristocrat and Vigen√®re solvers
- `app/data/spanish_quadgrams.txt` ‚Äî Spanish quadgram frequencies, used by the Xenocrypt solver

Source for these: practicalcryptography.com.
